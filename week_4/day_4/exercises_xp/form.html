Form Structure:

‚ùì Where will the sent data appear?

When you use the GET method, the form data appears in the browser's **URL**.

Example URL after submission:

form.html?user_name=John+Doe&user_message=Hello

The data is appended to the URL as query parameters, making it visible and limited in size.

üåü Exercise 2: HTML Form #2 (POST Method)

Form Structure (Same as Exercise 1):

‚ùì Where will the sent data appear?

When you use the POST method, the form data is packaged into the **Request Body** of the HTTP request.

Hint Answer: Network Tab

The data is **not visible in the URL**. You would need to open your browser's Developer Tools (F12), navigate to the **Network tab**, and inspect the details of the submission request to see the payload (Request Body).

This method is used for sensitive or large amounts of data.

Note: Since both forms submit to this same HTML file, the GET submission will reload the page with parameters in the URL, and the POST submission will result in a browser error (as this file is not a server capable of handling a POST request) but you will be able to inspect the POST data in the Network tab right before the error appears.
